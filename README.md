# MCAT Toy Demos

This repository is a static website for small MCAT study tools. It is designed to work on GitHub Pages with no build step, so the same files can be opened locally or from an iPad.

Start with `index.html`.

## Included tools

- `demos/adh-water-toy.html` - a simple causal toy showing how ADH changes collecting duct permeability, water reabsorption, urine volume, and urine concentration.

## Framework

The design principles for future toys are saved in `MCAT_TOY_FRAMEWORK.md`.
